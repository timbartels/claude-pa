# Code of Conduct

This project adopts the **Contributor Covenant v2.1** as its code of conduct.

Full text: https://www.contributor-covenant.org/version/2/1/code_of_conduct/

## Enforcement

Reports go to the maintainer (Tim Bartels) via the email listed in the GitHub profile, or by opening a confidential security advisory on the repository.

The maintainer reviews every report and responds within a reasonable timeframe. Outcomes follow the Community Impact Guidelines published alongside the Covenant.
